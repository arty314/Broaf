Broaf.Readme

개발환경: Android Studio Giraffe | 2022.3.1 Patch 2

실행 방법:
안드로이드 7.0 Nougat (API Level 24) 이상 지원, 함께 제출한 드라이브에서 apk파일을 다운로드 받아서 설치를 하시면 됩니다.

실행시 주의사항
- Broaf.apk 설치 시 보안 경고는 모두 무시해주세요.


- 데이터베이스로 Firebase를 사용하나 적용된 보안규칙이 전혀 없으므로 회원가입 테스트시 실제 정보를 입력하지 말아 주시기 바랍니다.
- 앱을 휴대폰에 설치하면 앱 아이콘이 2개가 나타나는데 둘중 아무거나 클릭을 하여도 똑같은 로그인 화면으로 연결됩니다.
- 권한을 거부하신 후 권한 요청 메세지가 다시 뜨지 않는다면 앱에 허용된 권한을 모두 해제하고 앱을 완전히 닫은후에 다시 시도해주기 바랍니다.
- 현재 개발이 진행중이므로 아직 구현이 되지 않은 기능이 많습니다. 내정보>계정정보 내에 있는 모든 버튼은 현재 구현된 기능이 없습니다.
- 회원가입과 게시글 작성은 RealtimeDatabase에 등록이 됩니다.

앱 사용방법
- 가장 첫화면에 앱 사용에 필요한 권한요구창과 로그인버튼, 회원가입버튼이 있습니다.
- 로그인버튼을 누르면 아이디(이메일 형식)와 비밀번호를 입력하는 칸이 있고 입력후 확인을 누르면 로그인을 하게 됩니다.
  하단 자동로그인은 개발시 편의를 위해 만든 버튼으로 테스터 계정으로 자동으로 로그인되게 됩니다.
- 회원가입 버튼을 누르면 계정에서 사용할 닉네임과 이메일(아이디로 사용), 비밀번호를 입력하고 확인을 누르면 회원가입이 끝나게 됩니다.

- 로그인에 성공할 시 지도가 보이는 메인화면으로 넘어가게 되고 화면구성은 다음과 같습니다.
  상단에 게시글 검색창이 있습니다.
  우상단에 있는 버튼 3개는 지도에 표시된 게시글 뱃지를 공개범위에 따라 필터해주는 버튼입니다.
  위에서부터 전체공개, 친구공개, 비공개(나에게만 보임)입니다.
  화면중앙에는 주변 게시글 아이콘과 위치가 표시됩니다. 하지만 실시간 위치추적은 현재 미구현 상태이고 우하단 새로고침버튼을 누르면 게시글검색창 아래에 텍스트로 사용자의 위도와 경도를 표시합니다.
  지도에 표시된 게시글 아이콘을 누르면 해당 게시글을 표시하는 뷰를 띄우게 됩니다. (현재 여기에 표시되는 게시글들은 Firebase와 연동이 안된 수동으로 입력한 데이터입니다.)
  좌측하단 위치버튼을 누르면 지도에 표시되어있는 위치로 화면이 이동하게 됩니다.
  하단 앱 네비게이션바에는 왼쪽에서부터 공지글을 볼 수 있는 공지탭으로 이동, 게시글 작성버튼, 계정정보를 볼 수 있는 계정정보탭으로 이동입니다.

- 검색창에 "사용자"라고 검색할 시 다른사용자~ 라는 게시글이 표시됩니다. 게시글을 누르면 게시글을 볼 수 있습니다. 또한 검색을 할 때에는 입력값이 최소 두글자이어야 합니다.
- 하단 중앙 게시글 작성 버튼은 메인 지도화면이 아닌 다른곳에 있을때는 메인화면으로 다시 돌아오는 버튼의 역할을 하게 됩니다. 그림도 지도모양으로 바뀌게 됩니다.
- 글쓰기 화면에서는 상단에 아이콘을 바꿀수도 있고 공개범위를 변경 할 수 있고, 공개 시간을 드래그해서 바꿀 수 있습니다. 현재 사진 첨부기능은 갤러리에서 첨부는 동작을 하나 데이터베이스에 저장되지는 않습니다. 카메라는 아예동작을 하지 않습니다. 하단 저장버튼을 누르면 데이터베이스에 입력한 정보가 등록되게 됩니다.
- 네이게이션바 우측 내 정보 버튼으로 들어온 내 정보 보기탭에서는 자신의 닉네임과 계정정보, 친구목록, 자신이 작성한 글 목록을 볼 수 있습니다. 계정정보창안에 있는 버튼들은 현재 아무 동작이 없습니다.
  친구목록은 자신과 친구를 맺은 사용자 목록이 표시됩니다. 닉네임을 입력한 후 +버튼을 누르면 친구가 추가됩니다.
  자신이 작성한 글 목록으로 글을 보면 뒤로가기 버튼이 동작하지 않습니다. 이때는 뒤로버튼을 누르면 나가집니다.
- 공지 탭에는 데이터베이스에 등록된 공지게시글이 표시됩니다.

담당파트
김동욱: 게시글 검색기능, 자신이 작성한 글 목록 보기
최지원: kakaomap api 사용하여 메인지도화면 구성, 지도상에 아이콘 표시 및 클릭시 게시글 표시, 전체 UI 구성
박남규: 친구추가/삭제 기능, 사용자 데이터타입 정의 및 Firebase와의 사용자 정보 연동
이승호: 게시글 작성및 Firebase에 게시글 등록, 메인화면 동작 보조
남은주: FirebaseAuth이용 로그인및 회원가입 구현, 인트로 화면구성및 유저 정보 등록
송진우: Firebase에 있는 정보를 표시해주는 앱내 리사이클러뷰 구현, 공지탭 구현
