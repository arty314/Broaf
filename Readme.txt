- 설치 시 뜨는 보안 경고는 모두 무시해주세요.

- 출처를 알 수 없는 앱 설치 > 무시하고 설치
- 안전하지 않은 앱 차단됨 > 세부정보 더보기 > 무시하고 설치하기

어플리케이션 실행 시 주의사항
- 데이터베이스로 Firebase를 사용하나 적용된 보안규칙이 전혀 없으므로 회원가입 테스트시 실제 정보를 입력하지 말아 주시기 바랍니다.
- 권한을 거부하신 후 권한 요청 메세지가 다시 뜨지 않는다면 앱에 허용된 권한을 모두 해제하고 앱을 완전히 닫은후에 다시 시도해주기 바랍니다.
- 현재 개발 중으로, 일부 기능은 아직 구현되지 않았습니다.
- 회원가입과 게시글 작성은 RealtimeDatabase에 등록이 됩니다.


앱 사용방법
- 가장 첫화면에 앱 사용에 필요한 권한요구창과 로그인버튼, 회원가입버튼이 있습니다.
- 로그인버튼을 누르면 아이디(이메일 형식)와 비밀번호를 입력하는 칸이 있고 입력후 확인을 누르면 로그인을 하게 됩니다.
- 회원가입 버튼을 누르면 계정에서 사용할 닉네임과 이메일(아이디로 사용), 비밀번호를 입력하고 확인을 누르면 회원가입이 끝나게 됩니다.

※ 만약 본인 계정으로 정상적인 로그인이 불가능할 경우 이메일: goodman@good.com   비밀번호: asdf123 계정으로 로그인을 시도해주십시오.


- 로그인에 성공할 시 지도가 보이는 메인화면이 표시됩니다.
   - 현재 위치 핀을 불러오기 위해서는 새로 고침을 누른 후 잠시 기다려주세요.
   - 과녁 버튼을 사용하기 위해서는 지도에 현재 위치 핀이 표시되어 있어야 합니다.

- 하단 바로 게시글을 작성하거나, 공지 혹은 내 정보 메뉴로 이동할 수 있습니다.

- 글쓰기 화면에서는 상단에 아이콘을 바꿀수도 있고 공개범위를 변경 할 수 있고, 공개 시간을 드래그해서 바꿀 수 있습니다.
   - 현재 사진 첨부기능은 갤러리에서 첨부는 동작을 하나 데이터베이스에 저장되지는 않습니다. 카메라는 동작 하지 않습니다.
   - 화면 하단의 저장 버튼을 누르면 데이터베이스에 입력한 정보가 등록됩니다.
- 내 정보 탭에서는 자신의 계정 정보와, 친구목록, 자신이 작성한 글 목록 등을 볼 수 있습니다.
- 친구 목록에는 자신이 친구로 등록한 사용자의 목록이 표시됩니다. 닉네임을 입력한 후 +버튼을 누르면 친구로 추가됩니다.
